Dash panel

S'il vous arrive d'oublier quelle fonction est sur certains switches, cette application peut vous aider

Utilisation

Installez l'application. Elle apparaitra dans le menu "Applications".

Vous pouvez choisir entre deux tailles d'affichage: petit (1 seul indicateur) et double (3 indicateurs)

Changez le nom, le switch, et le type d'indicateur. Pensez à appuyer sur "Sauv" après avoir fait les ajustements

Pour finir, allez dans "Chronos/Senseurs"->"Télémétrie Affichée", appuyez sur "Ajo.", selectionez "Lua" et selectionez le dashboard.
