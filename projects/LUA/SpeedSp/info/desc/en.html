Speed Announcer

Description

If a speed sensor is operational, this apps helps on announcing the speed without the units

Usage

Install the app. It will appear on the user application Menu.

Choose the speed sensor

Adjust the switch that will activate the speech.

The Min Speed set the speed below which the announces are exactly the current speed.

Above this speed, only tenths are announced (i.e. 90 instead of 91 or 92).

Pay special attention to units!!. Maybe your sensor does not report the speed on the standard Jeti units.

Standard units are (case sensitive):

  • m/s : Meters per second
  • km/h : Kilometers per hour
  • ft/s : Feets per second
  • mph : Miles per hour
  • kt. : Knots

    • If your sensor reports the speed on any other unit, the application will consider it is reporting on m/s
