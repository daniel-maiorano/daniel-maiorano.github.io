Annonces de vitesse

Description

Si un capteur de vitesse est installé, cette application aide à annoncer la vitesse sans les unités.

Usage

Installer l'application, elle apparaitra dans le menu Applications

Choisir le capteur de vitesse

Choisir l'inter qui active les annonces

Le paramètre "Vitesse minimum" établit la vitesse en dessous laquelle les annonces sont précisement la vitesse actuelle.

Au dessus de cette vitesse, uniquement les dizaines sont annoncés (par exemple 90 à la place de 91 ou 92).

Attention aux unités!! Peut être le capteur de vitesse n'utilise pas les unités standard Jeti

Les voici (majuscules et minusciles comptent):

  • m/s : Mètres par seconde
  • km/h : Kilomètres à l'heure
  • ft/s : Pieds par seconde
  • mph : Milles à l'heure
  • kt. : Noeuds

    • Si le capteur reporte la vitesse dans une unité autre que les précédentes, l'application considère qu'elle est en m/s.
